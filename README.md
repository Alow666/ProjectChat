# ProjectChat

Список участников команды: Дионисий(делал один)

Описание выбранной идеи решения: 
1)После раздумий над проектом было принято решение написать вектор с пользователями(классом где хранятся все их данные). 
2)Затем был написан класс Entrance который позволяет проверять логин и пароль пользователя на соответствие и давать разрешение на вход.
3)Следующие, что было прописано это вкладка друзья(без подтверждения запроса в друзья, не стал доделывать поскольку этого не было в требованиях проекта),
после чего была добавлена вкладка "Настройка аккаунта", где пользователю предоставляется возможность изменять логин, пароль, имя и фамилию.
4)Приватный чат между пользователей, долго думал как написать, но принял решение, что бы пользователю приходили все сообщения которые ему писали, затем он писал на выбор кому хочет.
5)Общий чат был реализован классом GeneralChat, в котором хранится 2 вектора с сообщением и индексом пользователя, по которому определяется кто писал.
  При каждом написанном сообщении пользователем или заходе во вкладку "Общий чат", чат обновляется.   
6)Чат поддерживает сообщения и данные о пользователе только на английском. Пытался сделать поддержку русского языка, но не получилось менял кодировку на 1251, пробовал через Windows.h, и setlocale. 

Описание пользовательских типов и функций в проекте:

	class User:
addFriends - добавляет друзей в вектор _friends;
getSizeFriends - возвращает размер вектора _friends;
getFriends - выводит на консоль имя и фамилию всех друзей в векторе _friends;
getInfoFriends - возвращает имя друга по индексу в векторе _friends;
checking_for_friends - проверяет на наличие друзей в векторе _friends;
getIndexFriens - возвращает индекс друга в векторе _friends;
createMessage - создание личного сообщения в вектор _chat;
sendMessage - отправление определенному пользователю сообщение;
getSizeChat - возвращает размер вектора _chat;
getIndexMessage - возвращает индекс отправителя сообщения в вектор _chat;
getTextMessage - выводит сообщение по индексу внутри вектора _chat;
getFriendsMesssage - выводит имя и имя пользователей в векторе _friends;
clearMessage - очищает вектор _chat после его чтения.

	class Entrance:
entranceLogin - проверка логина при входе в аккаунт;
entrancePassword - проверка пароля при входе в аккаунт(с ограниченными попытками).
